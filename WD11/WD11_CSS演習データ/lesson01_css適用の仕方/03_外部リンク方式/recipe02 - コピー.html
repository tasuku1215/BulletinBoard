ロースハムとおくらの冷やしラーメン

たっぷりのうす塩ロースハムとおくらなどの材料をトッピングした冷やしラーメンです。
おくらの独特の食感が冷たい麺とよく合います。

材料（1人分）

  • うす塩ロースハム　3枚
  • おくら　3本
  • レタス　1枚
  • きゅうり　1/4本
  • ゆでうずら卵　1個
  • ラディッシュ　1個
  • 中華麺　1玉
  • タレ（・酢:大さじ1　・ごま油：小さじ2/3　・しょうゆ：大さじ1/2）

作り方

  1. ロースハムは1/4に、おくらはゆでて水気を切り1/2にカットします。
    きゅうりはうす切り、ゆでうずら卵、ラディッシュは輪切りにします。
  2. たっぷりの湯で中華麺をゆで、冷水で洗って水気を切ります。
  3. 皿にちぎったレタスを敷き、中華麺を皿に盛ります。
    (1)のロースハム、おくら、きゅうり、ゆでうずら卵、ラディッシュを盛り付けてタレをかけます。
